# Julia CMO Website

Website profesional para Head of Marketing / CMO B2B SaaS.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React

## Design System

- **Aesthetic**: Swiss brutalism editorial + tech B2B SaaS
- **Colors**: White (#FFF), Near-black (#1A1A1A), Accent blue (#0066FF)
- **Typography**: Inter (sans-serif)
- **Grid**: 12 columnas, max-width 1280px

## Pages

- `/` - Homepage
- `/about` - Sobre mí, timeline, approach
- `/case-studies` - 4 casos de estudio detallados
- `/contact` - Email, Calendly, tipos de conversación

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static export se genera en `dist/`.

## Deploy

El sitio está configurado para exportación estática. Puedes deployar en:
- Vercel
- Netlify
- GitHub Pages
- Cualquier hosting estático
